## 安装

首先通过 NPM 安装 JSHint：

```sh
$ npm install -g jshint
```

然后通过 [Sublime Text Package Manager](https://sublime.wbond.net/) 安装 [JSHint Gutter](https://sublime.wbond.net/packages/JSHint%20Gutter)。
